# 🕋 Guide Complet du Hajj 2026

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

> وَأَذِّن فِي النَّاسِ بِالْحَجِّ يَأْتُوكَ رِجَالًا وَعَلَىٰ كُلِّ ضَامِرٍ يَأْتِينَ مِن كُلِّ فَجٍّ عَمِيقٍ

Guide interactif complet pour le pèlerinage du Hajj — bilingue Français/Arabe.

## 🌐 Démo

Ouvrez simplement `index.html` dans votre navigateur. Aucun serveur requis.

## ✨ Fonctionnalités

### 🕋 Mon Hajj — Guide jour par jour
- **5 jours détaillés** avec programme horaire, duas essentielles, alertes et conseils
- **Mini-carte SVG interactive** montrant votre position (Kaaba → Mina → Arafat → Muzdalifah → Jamarat)
- **Navigation par jour** avec boutons ◀ ▶ et noms de lieux

### 📖 Apprendre — Savoir religieux complet
- **22 étapes pas à pas** avec instructions détaillées
- **9 jours de timeline** du Hajj
- **10 rituels** avec illustrations SVG, histoire et règles
- **21 duas** en arabe calligraphié + translittération + français
- **Tenue/Ihram** avec SVG homme vs femme + interdits visuels
- **Pyramide Arkaan/Wajibaat/Sunan** (classification SVG)
- **3 Types de Hajj** (comparaison visuelle Tamattu'/Qiran/Ifrad)
- **Carte des distances** SVG (Kaaba → Mina → Arafat → Muzdalifah → Jamarat)

### ✅ Préparer — Concret et pratique
- **58 items checklist** interactive avec localStorage
- **Calendrier Mai 2026** avec intégration Google Calendar + Outlook + .ics
- **15 événements calendrier** avec horaires précis et 2 rappels (1h + 15min)
- **Coûts par pays** (7 pays, barres visuelles)
- **8 applications** recommandées (style App Store)
- **Guide Nusuk** (inscription officielle)

### 🔍 Plus — Tout le reste
- **22 FAQ** (accordéon)
- **Top 10 erreurs** (cartes avant/après ❌→✅)
- **16 conseils santé** (cartes colorées par type)
- **Survie chaleur** (thermomètre SVG animé, protocole 2024)
- **Accessibilité** (PMR, seniors, enceintes — 6 catégories, 31 items)
- **32 flashcards quiz** (flip animation)
- **Tour virtuel** des lieux saints
- **Guide Médine** (plan SVG du Rawdah)
- **12 questions pratiques** (situations tabous)
- **Liens externes** vers sites officiels

### 🌍 Bilingue FR / عربي
- **81 clés UI** traduites (tabs, boutons, labels, titres)
- **6 sections data** entièrement en arabe :
  - Guide du Compagnon (5 jours)
  - 10 Rituels
  - 22 FAQ
  - 16 Conseils Santé
  - 10 Erreurs courantes
  - 46 items Checklist

## 🚀 Utilisation

```bash
# Cloner le repo
git clone https://github.com/VOTRE-USERNAME/hajj-guide.git

# Ouvrir dans le navigateur
open index.html
# ou
python3 -m http.server 8000
```

Aucune dépendance. Aucun build. Aucun framework. **Pure HTML/CSS/JS.**
